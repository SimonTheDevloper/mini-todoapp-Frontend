# Welcome to Tickivo

Tickivo is a simple and minimalistic Todo-list with backend synchronisation.

## What can it do?

- you can login
- you can create a new account with your username your password and your email
- you can add todos
- you can read todos
- you can see wich todos are completed or open
- you can mark todos completed
- you can delete todos

## Tech Stacks

- plain HTML + Tailwind CSS and for Components Daisy UI
- plain JS
